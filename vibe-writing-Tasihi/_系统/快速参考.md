## ⚡ 快捷命令

| 命令 | 功能 | 使用场景 |
|------|------|----------|
| `/结构` | 进入结构规划阶段 | 有知识卡片，需要规划输出结构 |
| `/迭代` | 进入迭代优化阶段 | 有输出卡片，需要继续优化 |
| `/润色` | 润色当前卡片 | 语言风格调整 |
| `/成稿` | 生成最终文章 | 输出卡片完成，串联成文 |
| `/生成卡片` | 立即生成知识卡片 | 不等4轮，直接生成 |

## 🔄 常用操作

### 项目启动
- 说出主题："我想写关于XX"
- 或使用脚手架：`./_系统/模板/新建项目.sh "项目名"`

### 继续项目
- 说"继续[项目名]"（如：继续每日写作重塑大脑）
- 或给出项目文件路径

### 查询状态
- "列出所有卡片" - 查看已生成知识卡片
- "回顾一下进展" - 查看项目当前状态

## 📚 四个阶段

```
学习Agent ──→ 结构Agent ──→ 写作Agent ──→ 成稿Agent
  对话激发       全局规划       迭代优化       串联成稿
```

### 1️⃣ 学习阶段
- **输入**：对话、提问、讨论
- **输出**：知识卡片（每4轮自动生成）
- **触发**：新主题 / "继续对话"

### 2️⃣ 结构阶段
- **输入**：知识卡片
- **输出**：输出卡片首版 + 白板节点
- **触发**：`/结构` / "形成结构"

### 3️⃣ 写作阶段
- **输入**：输出卡片 + 用户反馈
- **输出**：优化后的输出卡片
- **触发**：`/迭代` / `/润色`

### 4️⃣ 成稿阶段
- **输入**：所有输出卡片
- **输出**：完整文章
- **触发**：`/成稿` / "最终输出"

## 📂 项目结构

```
项目/[项目名]/
├── 项目信息.md          # 进度跟踪
├── 初始文档.md          # 主题背景
├── 对话论述.md          # 对话记录（可选）
├── 内容白板.canvas      # Obsidian白板
├── 知识卡片/            # 学习阶段产出
├── 输出卡片/            # 结构+写作阶段产出
└── 最终成稿-xxx.md      # 成稿阶段产出
```

## 🎯 工作流速查

| 当前状态 | 下一步 | 命令 |
|----------|--------|------|
| 有主题 | 开始对话 | 直接说出主题 |
| 对话4轮+ | 生成卡片 | 自动触发 或 `/生成卡片` |
| 有知识卡片 | 规划结构 | `/结构` |
| 有输出卡片 | 继续优化 | `/迭代` 或 `/润色` |
| 卡片完成 | 生成文章 | `/成稿` |

## 🔧 路径规则

- **文件操作**：使用相对路径 `项目/`、`_系统/`
- **Obsidian双链**：使用完整路径 `{WORK_DIR}/项目/...`
- **Canvas节点**：使用完整路径 `{WORK_DIR}/项目/.../xxx.md`

## 💡 核心理念

> **Vibe Writing 是思维协同系统，不是AI写作助手**

- ✅ 你掌握：选择讲什么、怎么讲、保留什么
- ✅ AI协同：激发灵感、提供结构建议、润色语言
- ❌ 不是：AI自动生成，你只审核

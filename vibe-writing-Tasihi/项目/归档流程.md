# 项目归档流程

当项目完成后，按以下流程归档：

## 归档检查清单

- [ ] 最终成稿已完成
- [ ] 项目信息.md 状态标记为"已完成"
- [ ] 所有输出卡片已组织到白板
- [ ] 无待转化的知识卡片

## 归档步骤

1. **创建归档目录**
   ```bash
   mkdir -p "项目/_归档/$(date +%Y-%m)"
   ```

2. **移动项目**
   ```bash
   mv "项目/[项目名]" "项目/_归档/$(date +%Y-%m)/"
   ```

3. **更新本文件**
   在下方添加归档记录
